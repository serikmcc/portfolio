About Erik McCoy

I have been in the Web & Software world for 8+ years, and have enjoyed every minute of it. From leaving with a degree when I first began this journey, to building my knowledge and skillset over time with trial and error, I truly enjoy every facet that this career field offers.

I mainly work within the PHP language, harnessing both the pure code itself and the many frameworks it has built around it, and leverage the front-end power of jQuery / JavaScript to deliver functional, beautiful, and working projects that fit the clients needs to the maximum. From simple informational sites for college students, to completely custom E-Commerce platforms that have all the bells and whistles, you are hard-pressed to find me saying, "No..." on any project I take on.

I have continued to build my knowledge of a myriad of other languages along the way, and have truly enjoyed the opportunites they have opened for me. A cleaner list of the languages / technologies I am familiar with:

  • PHP Frameworks: CodeIgniter, CakePHP, Laravel
  • MySQL / SQLite
  • REST API / API development
  • jQuery / JavaScript
  • React.js
  • Java
  • Python
  • C# -- minimal and within the Unity engine landscape
